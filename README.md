# tinyChat #

current version: 0.0.3

## Description ##

### Tiny Chat app working with Node.js and Socket.io ###

## Installation ##

### npm install tinyChat ###

## Setup ##

1. Run:

		node server.js

1. Browse to:

		http://localhost:8080/index.html

1. Modify code as you want

1. Enjoy the tinyChat!
